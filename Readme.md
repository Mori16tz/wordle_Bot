# Wordle Discord Bot

A multi language discord bot that is played in the DM's of the discord bot. A user has 6 tries per language, per day. The streak is also counted.

![example image](https://i.imgur.com/We3J8lQ.png)

Currently supporting:
 - English
 - German

## Ussage

### /sprachauswahl or /language_selection

Allows the user to change the language.

### /info

Just the basic information on how the bot works.
